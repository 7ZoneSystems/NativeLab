v0.3.1

1) Fixed error surface in labs endpoints to show real endpoint errors instead of only HTTP 400 text.
2) Added py-to-doc project pause/resume checkpoints in localllm/temp for restart and shutdown recovery.
3) Moved Labs, Logs, Integrations, Pipeline, and MCP under a Dev tab with a contextual left sidebar, and moved Config to the top-right settings button.
4) Added Dev > Skills with a shared localllm skill library, /skills endpoint, and chat toggle for skill injection across chat, labs, and integrations.
5) Added built-in edit skill and structured-edit lab for temp-backed iterative code edits before saving to source files.
6) Added live structured-edit response capture and colored + / - line counts for each model edit.
7) Upgraded structured-edit into a chat-style edit workspace with saved checkpoints, branch-from-history, editable responses, clickable per-turn diffs, live file preview, and Python diagnostics.
8) Added Python diagnostics in Code Edit with syntax checking, lightweight warnings, a Problems panel, and editor line highlighting.
9) Moved detailed Code Edit diffs into per-message chat cards so users can inspect changes for every prompt in the conversation.
10) Fixed Dev tab and Dev sub-tab heading colors so they update correctly when switching dark/light themes.
11) Added independent scrolling to the Code Edit file sidebar so diagnostics, structure, logs, and the editor do not overlap on small screens.
12) Upgraded `nativelab --cli` into a terminal control center with Chat, Models, API Models, Skills, Labs, Pipelines, Integrations, Status, Setup, and Quit.
13) Added shared CLI runtime wiring so chat, Labs, pipelines, endpoint serving, integrations, local GGUF models, API models, and skill injection use the same backend surface.
14) Added CLI commands for models, api-models, skills, labs, pipeline, integrations, endpoint, and serve, with `--json` on inspect/list/show flows.
15) Expanded CLI chat slash commands with `/models`, `/api-models`, `/skills`, `/pipelines`, `/pipeline`, `/labs`, `/py-to-doc`, `/code-edit`, `/endpoint`, and `/serve`.
16) Added CLI model/API management for listing, loading, and setting defaults in `localllm/cli_prefs.json`.
17) Added CLI skill create/edit/delete/enable/disable controls and a saved chat skill toggle.
18) Added CLI Labs support for py-to-doc single/queue/project modes and temp-backed structured code-edit sessions with terminal diffs, save/save-as, and optional editable model JSON responses.
19) Added CLI saved pipeline list/show/run support and foreground integration endpoint serving.
20) Added CLI Discord/WhatsApp profile list/show/create/edit/delete/run commands with redacted secret display.
21) Refreshed CLI docs, beginner guide, README CLI section, and CLI smoke paths for clone/package usage.
