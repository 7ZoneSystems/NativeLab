# Security Policy

## Supported Versions

NativeLab (desktop) and PhonoLab (Android) are currently in active development. Only the latest version on main is supported.

| Project | Version | Supported |
|---------|---------|-----------|
| NativeLab | >= v0.2.5 | yes |
| NativeLab | latest | yes |
| PhonoLab | latest | yes |
| older | any | no |

## Reporting a Vulnerability

Do report security vulnerabilities through public GitHub issues.

You can also email the maintainer directly at 7zonesystems@zohomail.in with:

- A description of the vulnerability
- Steps to reproduce it
- Potential impact

You can expect an acknowledgement within 3 days. If the vulnerability is confirmed, a fix will be prioritized. If it is declined, you will receive an explanation.
